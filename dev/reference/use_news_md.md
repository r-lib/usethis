# Create a simple `NEWS.md`

This creates a basic `NEWS.md` in the root directory.

## Usage

``` r
use_news_md(open = rlang::is_interactive())
```

## Arguments

- open:

  Open the newly created file for editing? Happens in RStudio, if
  applicable, or via
  [`utils::file.edit()`](https://rdrr.io/r/utils/file.edit.html)
  otherwise.
