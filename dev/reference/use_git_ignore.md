# Tell Git to ignore files

Tell Git to ignore files

## Usage

``` r
use_git_ignore(ignores, directory = ".")
```

## Arguments

- ignores:

  Character vector of ignores, specified as file globs.

- directory:

  Directory relative to active project to set ignores
