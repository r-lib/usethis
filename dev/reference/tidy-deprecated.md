# Deprecated tidyverse functions

**\[deprecated\]**

- `use_tidy_style()` is deprecated because tidyverse packages are moving
  towards the use of [Air](https://posit-dev.github.io/air/) for
  formatting. See
  [`use_air()`](https://usethis.r-lib.org/dev/reference/use_air.md) for
  how to start using Air. To continue using the styler package, see
  `styler::style_pkg()` and `styler::style_dir()`.

## Usage

``` r
use_tidy_style(strict = TRUE)
```
